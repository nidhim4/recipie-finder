Recipe Finder

Introduction

Welcome to our website, RecipeFinder! Our website integrates several components to create a functional system for taking an input of ingredients and outputting recipes containing those ingredients. Our project uses CSS, Gradle, HTML, Java, JS, and SQL.

A web app built with HTML and JavaScript allows users to input an ingredient via a form, receiving a list of foods containing that ingredient. It uses two services: IngredientService and RecipeService, both employing HashMaps for quick data access. The frontend collects user input and sends it to the backend, where IngredientService finds matching foods using its HashMap. Meanwhile, RecipeService manages restaurant menus, highlighting the app's broad data handling capabilities. After processing, the backend returns a list of matching foods to the frontend for display, enhancing user experience through efficient data retrieval and structured data management.
